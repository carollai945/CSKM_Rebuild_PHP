# F03 資料庫備份 SDD

## 1. 對應需求

- Story：`F03-1`
- 對應 BDD：`bdd/f03-backup.feature`
- 目標：讓管理者手動執行資料庫備份並查看備份結果

## 2. 角色

- 系統管理員：啟動備份、查看備份紀錄
- 系統：執行備份命令並回報結果

## 3. 畫面與路由

- `F03.jsp`：備份入口與結果列表

## 4. 資料設計

- 主檔：`backup_log`
- 核心欄位：備份時間、檔名、執行者、結果、錯誤訊息

## 5. 流程設計

1. 管理者進入備份頁
2. 按下備份按鈕後系統執行備份作業
3. 完成後顯示結果並記錄 log

## 6. 權限與前端互動

- 僅管理者可見備份按鈕
- 頁面需明確顯示最近備份紀錄與成功/失敗結果

## 7. 錯誤處理

- 備份失敗需顯示錯誤訊息
- 非管理者不得啟動

## 8. 可測試點

- 備份啟動
- 成功/失敗紀錄
- 管理權限
