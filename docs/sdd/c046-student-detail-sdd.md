# C046 學員明細 SDD

## 1. 對應需求

- Story：`C04-2`
- 對應 BDD：`bdd/c046-student-detail.feature`
- 目標：集中檢視單一學員的基本資料、課程、服務與繳費摘要

## 2. 角色

- 教務/學顧：查看學員完整資料並進一步進入子功能
- 系統：整併多來源資料並提供跳轉入口

## 3. 畫面與路由

- `C046.jsp`：單一學員明細頁
- 關聯頁：`C041.jsp`、`C043.jsp`、`C045.jsp`

## 4. 資料設計

- 主檔：`student`
- 關聯：課程紀錄、服務紀錄、繳費紀錄
- 核心欄位：個資、就讀課程、學顧、狀態、最近服務與最近繳費

## 5. 流程設計

1. 從學員列表進入明細頁
2. 系統整併顯示個人資訊、課程與繳費摘要
3. 使用者可再進入對應子頁編修

## 6. 權限與前端互動

- 明細頁應提供返回列表與子頁跳轉
- 僅可見本人權限範圍內的學員資訊
- 關鍵狀態欄位需明確顯示

## 7. 錯誤處理

- 查無學員資料時應阻擋進入明細
- 無權限時不得顯示個資明細

## 8. 可測試點

- 明細整合顯示
- 跳轉子頁
- 資料權限控制
