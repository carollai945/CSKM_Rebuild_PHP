# A02 個人報表入口 SDD

## 1. 對應需求

- Story：`A02-1`
- 對應 BDD：`bdd/a02-personal-report.feature`
- 目標：讓員工查詢自己的日報與週報，並由入口頁進入新增、修改與檢視流程

## 2. 角色

- 員工：查詢自己的報表並進入明細
- 系統：依報表類型重建列表、帶入工作角色與報表種類

## 3. 畫面與路由

- `A02.jsp`：個人報表入口頁
- `POST /A/A02`：進入報表明細或新增流程
- `POST /A/A02/AjaxSearch`
- `POST /A/A02/AjaxSearchDefault`

## 4. 資料設計

- 主檔：個人日報/週報資料表
- 核心欄位：報表編號、日期、內容、類型、狀態、角色

## 5. 流程設計

1. 首次載入時若日期空白，先查最近五筆資料
2. 自動帶入迄日為今天、起日為七天前
3. 使用者輸入完整日期區間後可查詢自己的日報與週報
4. 系統依狀態顯示修改或檢視入口
5. 使用者可由四種新增按鈕帶入不同報表類型與角色

## 6. 權限與前端互動

- 日報與週報分成兩個列表區塊
- 行政與學顧類資料以不同顏色標示
- 搜尋時需帶入工作角色
- 修改/檢視按鈕以報表編號送出

## 7. 錯誤處理

- 起訖日期未完整輸入時不得查詢
- Ajax 失敗時顯示錯誤訊息

## 8. 可測試點

- 預設查詢
- 日期查詢
- 列表重建
- 新增入口角色切換
