# B00 電訪名單管理 SDD

## 1. 對應需求

- Story：`B00-1`
- 對應 BDD：`bdd/b00-lead-management.feature`
- 目標：以 Vue 頁面管理電訪名單查詢、新增、編修、派發與刪除

## 2. 角色

- 業務人員：查詢名單、建立/修改自己的名單、查看派發結果
- 區域主管：可批次派發、批次刪除、查看轄區資料
- 系統：維護 Vue 查詢狀態、選取清單與 modal 預設值

## 3. 畫面與路由

- `B00.jsp`：Vue 主頁，含查詢區、資料表、批次動作、編輯 modal
- 關聯頁：`B010.jsp`、`B011.jsp`

## 4. 資料設計

- 主檔：`lead`、`lead_trace`
- 核心欄位：名單編號、姓名、電話、來源、教育程度、區域、負責人、狀態、建立人

## 5. 流程設計

1. 頁面初始化時載入預設下拉與角色預設查詢條件
2. 使用者以 `v-model` 維護條件並重新查詢
3. 新增/編修共用 modal，開啟時先重設欄位或帶入原資料
4. 主管可批次勾選名單後派發或刪除

## 6. 權限與前端互動

- 每頁筆數調整後重新查詢
- 全選/全不選直接作用於 Vue 選取陣列
- 儲存按鈕須受必填欄位驗證控制
- 僅主管或原建立者可刪除單筆資料
- 教育程度為其他時需顯示自填欄位

## 7. 錯誤處理

- 缺少姓名、聯絡方式或必要來源不得儲存
- 未勾選資料不得執行批次派發/刪除
- 無權限者不得刪除他人資料

## 8. 可測試點

- 初始查詢與預設條件
- 新增/編修 modal 重置與帶值
- 單筆刪除權限
- 批次派發與批次刪除
