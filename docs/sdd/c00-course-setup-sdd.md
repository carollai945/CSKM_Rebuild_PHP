# C00 課程與主資料設定 SDD

## 1. 對應需求

- Story：`C00-1`
- 對應 BDD：`bdd/c00-course-setup.feature`
- 目標：維護課程、機構、科目與相關主資料設定

## 2. 角色

- 教務管理者：新增、修改課程與關聯設定
- 系統：提供階層式選單與資料一致性驗證

## 3. 畫面與路由

- `C00.jsp`：課程設定入口與查詢列表
- `C010.jsp`：課程或主資料編修頁

## 4. 資料設計

- 主檔：`course`、`institution`、`subject`
- 核心欄位：區域、分校、課程名稱、期別、上課日、狀態、費用設定

## 5. 流程設計

1. 使用者先選區域/機構，再選課程條件
2. 查詢結果顯示可編修課程主檔
3. 進入明細頁新增或修改課程與相關設定

## 6. 權限與前端互動

- 區域、機構、課程採連動下拉
- 列表與編輯頁需反映啟用/停用狀態
- 僅具教務設定權限者可儲存變更

## 7. 錯誤處理

- 缺少主資料或重複課程代碼不得儲存
- 無權限者不得修改

## 8. 可測試點

- 連動下拉
- 課程新增/修改
- 啟用狀態控制
- 權限限制
