# E02 請款清單 SDD

## 1. 對應需求

- Story：`E02-1`
- 對應 BDD：`bdd/e02-invoice-list.feature`
- 目標：提供財務查看請款單列表、進入明細並執行後續處理

## 2. 角色

- 財務人員：查詢請款清單、查看狀態、進入處理頁
- 系統：呈現請款狀態與關聯資料

## 3. 畫面與路由

- `E02.jsp`：請款列表頁
- `E020.jsp`：請款明細頁

## 4. 資料設計

- 主檔：`invoice_request`
- 核心欄位：請款單號、申請人、請款項目、金額、狀態、送件日期、關聯簽呈

## 5. 流程設計

1. 財務於列表查詢請款資料
2. 進入明細查看附件與請款內容
3. 視狀態導向確認或退回流程

## 6. 權限與前端互動

- 列表需顯示狀態與明細入口
- 明細頁保留附件、關聯簽呈與返回清單操作
- 僅財務角色可進入處理流程

## 7. 錯誤處理

- 查無資料時應顯示空清單
- 無權限者不得進入明細

## 8. 可測試點

- 請款列表查詢
- 明細導頁
- 財務角色限制
