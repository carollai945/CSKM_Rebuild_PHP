# 財務報表與請款流程 SDD

## 對應 Story

- E01-1、E02-1、E03-1

## 畫面與路由

- `/E/E01`、`/E/E01/AjaxSearch`
- `/E/E02`、`/E/E02/AjaxSelect`、`/E/E02/AjaxSearch`
- `/E/E03`、`/E/E03/AjaxSearch`、`/E/E03/Finance`、`/E/E03/returnBack`

## 設計重點

- E01：年度收入報表，依年份與小數位設定查詢
- E02：個人或請款清單檢索，支援日期與部門條件
- E03：財務確認與退件

## 資料

- `payment_info`
- 請款相關主檔
- 財務確認狀態 / 退件原因

## 權限

- 財務角色可最終確認或退件
- 業務角色在 E02 僅查自己範圍資料

## 測試切點

- 報表查詢
- 小數位與年度篩選
- 請款確認 / 退件
