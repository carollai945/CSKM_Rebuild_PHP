# F05 人員總覽 SDD

## 1. 對應需求

- Story：`F05-1`
- 對應 BDD：`bdd/f05-staff-overview.feature`
- 目標：查詢人員清單、顯示基本任職資訊與權限摘要

## 2. 角色

- 系統管理員：查詢所有人員
- 區域主管：查看轄區人員
- 系統：依角色控制清單範圍

## 3. 畫面與路由

- `F05.jsp`：人員列表與查詢條件

## 4. 資料設計

- 主檔：`hr_management`
- 關聯：`department`、`title`、`menu`
- 核心欄位：員工編號、姓名、區域、部門、職稱、在職狀態、權限摘要

## 5. 流程設計

1. 使用者設定查詢條件
2. 系統回傳可見範圍內的人員清單
3. 需要時再導向 F02 進行權限調整

## 6. 權限與前端互動

- 管理者可看全部；主管僅看本區
- 列表需顯示在職狀態與明細/編修入口

## 7. 錯誤處理

- 無權限者不得查看超出範圍資料
- 查無資料時應顯示空清單

## 8. 可測試點

- 條件查詢
- 角色資料範圍
- 人員狀態顯示
