# E01 收入報表 SDD

## 1. 對應需求

- Story：`E01-1`
- 對應 BDD：`bdd/e01-income-report.feature`
- 目標：提供財務人員依日期、區域與類別查詢收入統計

## 2. 角色

- 財務人員：查詢與檢視收入報表
- 管理者：查看彙總結果

## 3. 畫面與路由

- `E01.jsp`：收入報表查詢頁

## 4. 資料設計

- 主檔：`payment_info`
- 關聯：課程、機構、申請單
- 核心欄位：收款日期、金額、收款類別、區域、課程、經辦人

## 5. 流程設計

1. 使用者設定查詢期間與篩選條件
2. 系統彙整收入資料並顯示統計結果
3. 使用者可依欄位檢視明細彙總

## 6. 權限與前端互動

- 日期條件與查詢按鈕為主要入口
- 可依角色限制可見區域或資料範圍
- 列表應同步顯示總額與明細

## 7. 錯誤處理

- 查詢期間不合法不得送出
- 無資料時應顯示空結果而非錯誤

## 8. 可測試點

- 條件查詢
- 金額彙總
- 權限資料範圍
