# F04 電訪名單匯入 SDD

## 1. 對應需求

- Story：`F04-1`
- 對應 BDD：`bdd/f04-import-leads.feature`
- 目標：透過 Excel 匯入電訪名單並回報成功/失敗筆數

## 2. 角色

- 系統管理員/主管：上傳匯入檔案
- 系統：解析 Excel、驗證欄位並寫入名單資料

## 3. 畫面與路由

- `F04.jsp`：匯入頁、檔案上傳與結果區

## 4. 資料設計

- 主檔：`lead`
- 匯入欄位：姓名、電話、來源、區域、備註等

## 5. 流程設計

1. 使用者選擇 Excel 檔案上傳
2. 系統驗證欄位格式與必要欄位
3. 寫入成功資料並列出失敗原因

## 6. 權限與前端互動

- 頁面需顯示匯入格式說明
- 上傳後回顯成功筆數、失敗筆數與錯誤列

## 7. 錯誤處理

- 檔案格式錯誤不得匯入
- 必填欄位缺漏須列出行號與原因
- 無權限者不得上傳

## 8. 可測試點

- 檔案上傳
- 欄位驗證
- 成功/失敗回饋
