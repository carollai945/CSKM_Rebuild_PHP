# C00 課程與主資料設定 頁面設計

## 1. 基本資料

- 頁面：`C00.jsp`
- 對應 BDD / SDD / TDD：`c00-course-setup`
- 使用者：教務管理者

## 2. 頁面目的

- 維護課程、機構、科目與相關主資料設定

## 3. 版面區塊

- 主選擇列：區域選擇、機構選擇、課程選擇
- 機構維護區：`orgTb`，含機構列表、輸入框與新增 / 儲存 / 刪除
- 課程維護區：`courseTb`，含課程列表、輸入框與新增 / 儲存 / 刪除
- 科目維護區：`subjectTb`，含科目列表、輸入框與新增 / 儲存 / 刪除
- 學員數顯示區：`showStudentDiv` 顯示目前課程人數

## 4. 欄位與顯示規則

- 區域、機構、課程採連動下拉
- 未選上層項目時，下層維護區會隱藏或重置
- 每個維護區底部都有對應的設定輸入框

## 5. 按鈕與動作

- `新增/修改 機構`：切換到機構維護區
- `新增/修改 課程`：切換到課程維護區
- `新增/修改 科目`：切換到科目維護區
- 各維護區的 `新增 / 儲存 / 刪除`：對目前清單執行維護

## 6. 互動規則

- 變更區域會刷新機構與課程選單
- 變更機構會刷新課程與對應資料表
- 新增或刪除前會先檢查是否選到必要上層節點與輸入內容

## 7. 訊息設計

- 主資料缺漏
- 課程代碼重複
- 儲存成功

## 8. 權限與狀態

- 僅具教務設定權限者可維護
