# A00 個人資料維護 頁面設計

## 1. 基本資料

- 頁面：`A00.jsp`
- 對應 BDD / SDD / TDD：`a00-personal-data`
- 使用者：員工本人、由 `F02` 帶入唯讀檢視的管理者

## 2. 頁面目的

- 提供員工維護個人基本資料、聯絡資料、語言技能、證照、家庭與工作經歷

## 3. 版面區塊

- 頁首與識別區：頁面標題、員工照片、員工基本識別資訊
- 基本資料表單區：姓名、性別、血型、生日、身分證、婚姻、兵役、聯絡方式
- 地址區：戶籍地址、通訊地址與 `同上` 複製按鈕
- 語言能力區：英日語能力與 `Language Skills` 表格欄位
- 技能區：一般軟體 / 程式語言、專業技能、自評能力
- 證照區：英文檢定、日文檢定、其他專業證照
- 家庭狀況區：家庭成員、關係、聯絡方式
- 工作經歷區：多段工作經歷表格
- 操作區：閱讀員工守則、儲存

## 4. 欄位與顯示規則

- 日期欄位使用 datepicker，例如生日與各段工作期間
- 多數聯絡欄位、數值欄位與證照欄位有長度或格式限制
- `同上` 會把戶籍地址內容直接帶入通訊地址
- 唯讀模式下保留資料顯示，但隱藏選單、按鈕與可編修輸入

## 5. 按鈕與動作

- `同上`：複製戶籍地址到通訊地址
- `閱讀員工守則`：另開 `staff.pdf`
- `儲存`：提交整份個人資料與照片

## 6. 互動規則

- 手機、體重、薪資等欄位需通過前端數字檢查
- 欄位驗證失敗時以 alert 提示，並停留在錯誤欄位
- 儲存成功後顯示成功訊息並重新載入；失敗則提示聯絡系統管理員

## 7. 訊息設計

- 儲存成功：顯示 `Save Success!`
- 儲存失敗：顯示 `Save Failed, please contact administrator!`
- 欄位錯誤：依欄位提示格式、長度或必填問題

## 8. 權限與狀態

- 一般員工僅維護自己的資料
- 從 `F02` 帶入時可進入唯讀檢視模式
