# F04 電訪名單匯入 頁面設計

## 1. 基本資料

- 頁面：`F04.jsp`
- 對應 BDD / SDD / TDD：`f04-import-leads`
- 使用者：系統管理員、主管

## 2. 頁面目的

- 上傳 Excel 匯入電訪名單，並回饋成功與失敗結果

## 3. 版面區塊

- 格式說明區
- 檔案上傳區
- 匯入結果區：成功筆數、失敗筆數、錯誤列

## 4. 欄位與顯示規則

- 需顯示可接受檔案格式與必要欄位
- 結果區需標示失敗行號與原因

## 5. 按鈕與動作

- `選擇檔案`
- `上傳匯入`

## 6. 互動規則

- 上傳後解析 Excel 並回傳逐列結果
- 部分失敗時仍保留成功匯入資料

## 7. 訊息設計

- 檔案格式錯誤
- 欄位缺漏
- 匯入成功 / 部分失敗

## 8. 權限與狀態

- 僅授權角色可上傳
