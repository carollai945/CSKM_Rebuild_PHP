# F05 人員總覽 頁面設計

## 1. 基本資料

- 頁面：`F05.jsp`
- 對應 BDD / SDD / TDD：`f05-staff-overview`
- 使用者：系統管理員、區域主管

## 2. 頁面目的

- 查詢人員清單並查看任職資訊與權限摘要

## 3. 版面區塊

- 查詢區：姓名、員工編號、區域、部門、狀態
- 列表區：人員摘要、在職狀態、權限入口

## 4. 欄位與顯示規則

- 列表顯示員工編號、姓名、區域、部門、職稱、在職狀態

## 5. 按鈕與動作

- `搜尋`
- `查看明細 / 編修權限`

## 6. 互動規則

- 依查詢條件刷新列表
- 由列表可跳轉至 F02

## 7. 訊息設計

- 查無資料
- 無權限查看全部資料

## 8. 權限與狀態

- 管理者可看全部
- 區域主管僅看本區
