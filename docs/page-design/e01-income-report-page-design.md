# E01 收入報表 頁面設計

## 1. 基本資料

- 頁面：`E01.jsp`
- 對應 BDD / SDD / TDD：`e01-income-report`
- 使用者：財務人員、管理者

## 2. 頁面目的

- 依期間與條件查詢收入統計與明細

## 3. 版面區塊

- 查詢區：日期、區域、類別
- 報表摘要區：總額、筆數
- 明細列表區

## 4. 欄位與顯示規則

- 必填條件為查詢期間
- 明細區顯示收款日期、類別、區域、金額

## 5. 按鈕與動作

- `查詢`
- `重設`

## 6. 互動規則

- 查詢成功後同步刷新摘要與明細
- 依角色限制可見區域

## 7. 訊息設計

- 日期區間錯誤
- 查無資料

## 8. 權限與狀態

- 財務可看授權範圍，管理者可看彙總
