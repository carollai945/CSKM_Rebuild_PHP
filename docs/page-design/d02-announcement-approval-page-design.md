# D02 公告審核 頁面設計

## 1. 基本資料

- 頁面：`D02.jsp`、`D020.jsp`
- 對應 BDD / SDD / TDD：`d02-announcement-approval`
- 使用者：主管、CEO

## 2. 頁面目的

- 依公告審核階段完成核准或退回

## 3. 版面區塊

- 待審列表區
- 公告明細區
- 審核意見區
- 操作區：核准、退回、返回

## 4. 欄位與顯示規則

- 依角色顯示不同審核按鈕
- 內容區保留公告主旨、內容、附件與引用來源摘要

## 5. 按鈕與動作

- `查看明細`
- `核准`
- `退回`

## 6. 互動規則

- 退回必填意見
- 審核完成後返回待審列表

## 7. 訊息設計

- 退回意見未填
- 核准成功 / 退回成功
- 無權限操作

## 8. 權限與狀態

- 僅能看到自己階段待審公告
