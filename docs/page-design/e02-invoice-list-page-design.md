# E02 請款清單 頁面設計

## 1. 基本資料

- 頁面：`E02.jsp`、`E020.jsp`
- 對應 BDD / SDD / TDD：`e02-invoice-list`
- 使用者：財務人員

## 2. 頁面目的

- 查詢請款單列表，查看明細並進入後續處理流程

## 3. 版面區塊

- 清單查詢區
- 請款列表區
- 明細區：金額、事由、附件、關聯簽呈

## 4. 欄位與顯示規則

- 列表顯示單號、申請人、金額、狀態
- 明細頁保留附件與關聯資料摘要

## 5. 按鈕與動作

- `搜尋`
- `查看明細`
- `進入處理`

## 6. 互動規則

- 由清單帶單號進入明細
- 視狀態導向 E03 或檢視模式

## 7. 訊息設計

- 查無資料
- 無權限操作

## 8. 權限與狀態

- 僅財務角色可查看與後續處理
