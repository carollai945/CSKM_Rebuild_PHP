# E03 財務確認 頁面設計

## 1. 基本資料

- 頁面：`E03.jsp`
- 對應 BDD / SDD / TDD：`e03-finance-confirm`
- 使用者：財務人員

## 2. 頁面目的

- 對請款單執行核准、退回與狀態追蹤

## 3. 版面區塊

- 查詢條件區：日期區間、部門、狀態與查詢按鈕
- 請款表格區：`E03_table`，列內容與操作按鈕由 `AjaxSearch` 後動態 append
- 退回 modal：`returnBack`，含請款單編號、退回原因、Close / Send

## 4. 欄位與顯示規則

- 清單顯示請款單號、日期、申請人、部門、用途說明、支付方式、金額與確認欄
- 用途說明欄會以紅字顯示自行退回原因
- 退回 modal 開啟時先帶單號並清空前次原因
- modal 標題中的請款單號由 `showModalNumber()` 動態回填

## 5. 按鈕與動作

- `查詢 Search`：重新整理待確認清單
- `Confirm View!`：完成財務確認 / 歸檔
- `return back`：開啟退回 modal
- `Send`：送出退回原因

## 6. 互動規則

- 查詢會先清空舊表格列，再重建新結果
- 退回需透過 modal 填寫原因
- 確認或退回成功後會 alert 並重新查詢
- 每列 `Confirm View!` 與 `return back` 按鈕都由前端依資料狀態動態組成

## 7. 訊息設計

- 退回原因未填
- 核准成功
- 退回成功

## 8. 權限與狀態

- 僅財務角色可操作待確認資料
