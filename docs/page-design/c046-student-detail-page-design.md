# C046 學員資料修改 頁面設計

## 1. 基本資料

- 頁面：`C046.jsp`
- 對應 BDD / SDD / TDD：`c046-student-detail`
- 使用者：教務、學顧

## 2. 頁面目的

- 維護單一學員的基本資料、聯絡方式、所屬課程與學顧資訊

## 3. 版面區塊

- 學員基本資料區：姓名、性別、區域、電話、手機、傳真、地址
- 個人背景區：學歷、生日、職業、Email、公司資料
- 課程與學顧區：機構、可複選課程、已選課程清單、學顧、來源、等級
- 建檔資訊區：建檔日期、下次聯繫日期、備註
- 操作區：確認儲存、返回列表

## 4. 欄位與顯示規則

- 課程欄位為多選清單，右側同步顯示已選課程清單
- 生日、建檔日與下次聯繫日使用 datepicker
- 姓名、手機等必填欄位以星號標示

## 5. 按鈕與動作

- `確認 Save`：提交修改後資料
- `返回列表 Back`：回到 `C04`

## 6. 互動規則

- 由 `C04` 帶入 `std_no` 後載入現有資料
- 課程多選變更時即時重建已選課程清單
- 儲存前需先通過欄位驗證

## 7. 訊息設計

- 必填欄位未完成
- 日期或格式錯誤
- 儲存成功 / 無法儲存

## 8. 權限與狀態

- 僅可修改授權範圍內學員資料
