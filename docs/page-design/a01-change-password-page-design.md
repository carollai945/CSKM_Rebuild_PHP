# A01 修改密碼 頁面設計

## 1. 基本資料

- 頁面：`A01.jsp`
- 對應 BDD / SDD / TDD：`a01-change-password`
- 使用者：已登入員工

## 2. 頁面目的

- 驗證舊密碼後更新個人登入密碼

## 3. 版面區塊

- 帳號顯示區：顯示目前登入帳號
- 密碼輸入區：舊密碼、新密碼、再次確認新密碼
- 操作按鈕區：確認、返回

## 4. 欄位與顯示規則

- 帳號欄位唯讀顯示
- 三個密碼欄位皆為密碼遮罩輸入
- 驗證失敗時保留本次輸入流程並顯示結果訊息

## 5. 按鈕與動作

- `確定`：送出舊密碼驗證與新密碼更新
- `返回`：回到上一頁或離開修改流程

## 6. 互動規則

- 系統需先比對舊密碼
- 兩次新密碼需一致才可更新
- 成功後新密碼立即生效

## 7. 訊息設計

- 舊密碼錯誤
- 兩次新密碼不一致
- 密碼更新成功

## 8. 權限與狀態

- 僅目前登入者可修改自己的密碼
