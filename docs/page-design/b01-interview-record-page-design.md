# B01 電訪紀錄 頁面設計

## 1. 基本資料

- 頁面：`B01.jsp`、`B010.jsp`、`B011.jsp`
- 對應 BDD / SDD / TDD：`b01-interview-record`
- 使用者：業務人員、主管

## 2. 頁面目的

- 建立與追蹤電訪結果，安排後續聯繫時間

## 3. 版面區塊

- 查詢列表區：待追蹤名單、結果、下次聯繫時間
- 明細區：聯絡結果、備註、下次追蹤、負責人

## 4. 欄位與顯示規則

- 列表顯示名單、追蹤結果與下次追蹤時間
- 明細頁保留前次紀錄供參考

## 5. 按鈕與動作

- `搜尋`
- `新增 / 更新紀錄`
- `查看明細`

## 6. 互動規則

- 聯絡結果與下次追蹤時間為核心欄位
- 送出後回寫列表狀態與下一節點

## 7. 訊息設計

- 聯絡結果必填
- 下次追蹤時間不合法
- 儲存成功

## 8. 權限與狀態

- 業務維護自己負責資料
- 主管可查看團隊或本區結果
