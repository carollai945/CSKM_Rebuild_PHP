# D04 學員意見管理 頁面設計

## 1. 基本資料

- 頁面：`D04.jsp`
- 關聯明細頁：`D040.jsp`
- 對應 BDD / SDD / TDD：`d04-student-feedback`
- 使用者：管理人員、服務人員

## 2. 頁面目的

- 查詢學員意見列表、篩選處理狀態，並進入明細頁追蹤回覆紀錄

## 3. 版面區塊

- 查詢條件區：起訖日期、區域下拉、部門下拉、處理狀態下拉（全部 / 待處理 / 處理中 / 已結案）、人員姓名欄位、查詢按鈕
- 意見列表表格區：每列顯示學員姓名、提交日期、區域、部門、處理狀態、負責人員與檢視按鈕
- 明細展開區（D040）：顯示意見內容、回覆紀錄清單與相關人員資訊

## 4. 欄位與顯示規則

- 列表查無資料時顯示「--」
- 處理狀態以標籤顯示：`PENDING`→ 待處理、`IN_PROGRESS`→ 處理中、`CLOSED`→ 已結案
- 明細區顯示意見全文、各回覆的回覆者與回覆時間
- 無回覆紀錄時顯示「尚無回覆」

## 5. 按鈕與動作

- `查詢 Search`：依條件重建意見列表
- `檢視 View`：展開 D040 明細覆蓋層，顯示意見明細與回覆紀錄
- `關閉`：關閉 D040 明細覆蓋層

## 6. 互動規則

- Ajax 查詢前會先清空舊的列表資料
- 一般使用者查詢時後端自動依授權範圍過濾
- 主管 / CEO 可跨區域查詢

## 7. 訊息設計

- 查詢失敗時顯示「查詢失敗，請稍後再試」
- 無回覆紀錄時顯示「尚無回覆」

## 8. 權限與狀態

- 一般使用者：僅能查看授權範圍（指定區域）資料
- 主管 / CEO：可查看所有區域資料
