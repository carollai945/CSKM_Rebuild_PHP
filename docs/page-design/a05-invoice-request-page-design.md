# A05 請款申請 頁面設計

## 1. 基本資料

- 頁面：`A05.jsp`、`A050.jsp`
- 對應 BDD / SDD / TDD：`a05-invoice-request`
- 使用者：申請人

## 2. 頁面目的

- 建立請款資料、附上說明與附件，送入財務確認流程

## 3. 版面區塊

- 歷史列表區：查詢條件、狀態、退回原因
- 明細區：請款項目、金額、說明、附件
- 操作區：儲存、送審、返回

## 4. 欄位與顯示規則

- 草稿與退回單可再次編修
- 已送出或處理中單據僅可檢視
- 列表顯示狀態與退回原因摘要

## 5. 按鈕與動作

- `建立`、`儲存`、`送審`
- `查看明細`
- `補件後重送`

## 6. 互動規則

- 金額與附件先做前端檢查
- 送審後資料進入財務處理清單

## 7. 訊息設計

- 金額格式錯誤
- 缺少必要附件
- 送審成功 / 退回待補件

## 8. 權限與狀態

- 僅申請人可修改自己的草稿或退回單
