# A06 公告管理 頁面設計

## 1. 基本資料

- 頁面：`A06.jsp`、`A060.jsp`、`A061.jsp`
- 對應 BDD / SDD / TDD：`a06-announcement`
- 使用者：公告建立者、主管、CEO

## 2. 頁面目的

- 建立一般公告或引用已核准簽呈後送審發布

## 3. 版面區塊

- 列表區：歷史公告、查詢條件、建立入口
- 編輯區：主旨、內容、發布範圍、附件、引用簽呈區
- 檢視區：公告內容、附件與審核狀態

## 4. 欄位與顯示規則

- 一般模式可直接輸入主旨與內容
- 引用簽呈模式僅能選本人已核准簽呈
- 引用後主旨與內容改為唯讀

## 5. 按鈕與動作

- `建立公告`
- `引用簽呈`
- `送審`
- `查看明細`

## 6. 互動規則

- 引用簽呈後隱藏一般送審按鈕
- 引用送出前需再次確認
- 列表顯示建立者、狀態與入口

## 7. 訊息設計

- 不可引用未核准或非本人簽呈
- 主旨或內容未填
- 送審成功

## 8. 權限與狀態

- 建立者可編修草稿或退回公告
- 審核者於 D02 處理後回寫狀態
