# F00 區域管理 頁面設計

## 1. 基本資料

- 頁面：`F00.jsp`、`F000.jsp`
- 對應 BDD / SDD / TDD：`f00-area-management`
- 使用者：系統管理員

## 2. 頁面目的

- 維護區域主檔與啟用狀態，供組織結構與資料範圍控管使用

## 3. 版面區塊

- 查詢列表區：顯示全部區域資料
- 編修區：輸入區域代碼、區域名稱、狀態
- 操作區：新增、編修、儲存

## 4. 欄位與顯示規則

- 列表欄位：區域代碼、區域名稱、狀態
- 區域代碼：必填，建立後不可重複
- 區域名稱：必填
- 狀態：啟用 / 停用
- 列表預設顯示最新儲存結果

## 5. 按鈕與動作

- `新增`：建立新區域資料
- `編修`：載入選取區域到編修區
- `儲存`：送出新增或編修資料

## 6. 互動規則

- 儲存前檢查必填欄位與區域代碼唯一性
- 儲存成功後清單即時刷新並保留最新狀態
- 非系統管理員不得看到新增、編修、儲存操作

## 7. 訊息設計

- 區域代碼重複
- 必填欄位未填寫
- 儲存成功
- 權限不足

## 8. 權限與狀態

- 僅系統管理員可操作
- 區域狀態遵循啟用 / 停用兩種顯示與儲存值
