# B02 學員服務管理 頁面設計

## 1. 基本資料

- 頁面：`B02.jsp`
- 關聯頁面：`B020.jsp`、`B021.jsp`、`B023.jsp`
- 對應 BDD / SDD / TDD：`b02-student-service`
- 使用者：業務、學顧、主管

## 2. 頁面目的

- 查詢已簽約學員，並由入口頁分流到學員意見、繳費紀錄與修課狀況子頁

## 3. 版面區塊

- 查詢條件區：區域、機構、課程、姓名與查詢按鈕
- 搜尋結果區：`searchResult` 學員列表，列內容由 Ajax 結果動態 append
- 功能入口欄位區：每列由 JavaScript 動態組出 `檢視繳費`、`檢視修課`、`檢視意見`
- 關聯子頁區：由入口頁帶入學員編號後進入 B020 / B021 / B023

## 4. 欄位與顯示規則

- 入口頁列表顯示學員姓名、手機、電話、Email 與三個功能按鈕
- 區域、機構、課程採連動下拉
- Ajax 搜尋前會清空舊結果後再重建表格
- 課程下拉選單內容會依機構重新以 Ajax 回填

## 5. 按鈕與動作

- `查詢 Search`：載入符合條件的已簽約學員
- `檢視意見`：進入 `B020`
- `檢視繳費`：進入 `B021`
- `檢視修課`：進入 `B023`

## 6. 互動規則

- 變更區域或機構時，會同步刷新下層選單
- 搜尋成功後以 JavaScript 動態建立列表列與功能按鈕
- 入口頁只負責查詢與分流，編修行為發生在關聯子頁
- 每列按鈕都在前端直接組入對應子頁的 hidden `std_no` 與 submit form

## 7. 訊息設計

- 未選課程或付款項目
- 金額格式錯誤
- 儲存成功

## 8. 權限與狀態

- 僅能維護授權範圍內學員
- 不同角色看見不同子頁與按鈕
