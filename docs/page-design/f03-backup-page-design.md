# F03 資料庫備份 頁面設計

## 1. 基本資料

- 頁面：`F03.jsp`
- 對應 BDD / SDD / TDD：`f03-backup`
- 使用者：系統管理員

## 2. 頁面目的

- 手動執行資料庫備份並查看最近備份結果

## 3. 版面區塊

- 操作區：備份按鈕
- 結果區：最近備份紀錄、成功 / 失敗狀態

## 4. 欄位與顯示規則

- 顯示備份時間、檔名、執行者、結果

## 5. 按鈕與動作

- `開始備份`

## 6. 互動規則

- 送出後執行備份並刷新結果列表

## 7. 訊息設計

- 備份成功
- 備份失敗與錯誤訊息

## 8. 權限與狀態

- 僅管理者看得到操作按鈕
