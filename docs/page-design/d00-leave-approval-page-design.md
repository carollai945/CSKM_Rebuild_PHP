# D00 請假審核 頁面設計

## 1. 基本資料

- 頁面：`D00.jsp`
- 關聯明細頁：`D000.jsp`
- 對應 BDD / SDD / TDD：`d00-leave-approval`
- 使用者：區域主管、CEO

## 2. 頁面目的

- 在待審清單中完成單筆或批次請假審核

## 3. 版面區塊

- 查詢條件區：日期區間、區域、部門、申請人與查詢按鈕
- 待審表格區：`D00_table`，列內容與檢視按鈕由 `AjaxSearch` 回傳後動態 append，同時含主管與 CEO 待回覆欄位
- 選取控制區：主管全選、CEO 全選、全不選
- 主管批次回覆區：主管意見 textarea 與 `回覆 Reply`
- CEO 批次回覆區：執行長批示 textarea 與 `回覆 Reply`
- 關聯明細審核頁：單筆檢視與核准 / 退回

## 4. 欄位與顯示規則

- 表格固定顯示假單編號、申請人、區域、部門、日期、事由、主管意見、CEO 核示
- 主管與 CEO 的勾選欄位分開顯示
- 查無資料時顯示 `No Result`
- 部門下拉選單會依區域切換後動態重建

## 5. 按鈕與動作

- `搜尋`
- `全選 / 全不選`
- `批次回覆`
- `核准 / 退回`

## 6. 互動規則

- 主管勾選時會取消 CEO 勾選；CEO 勾選時也會取消主管勾選
- 批次回覆依角色送出不同 type 值
- 失敗時會提示批次回覆錯誤的假單編號
- 搜尋結果會由 JavaScript 依狀態動態決定勾選欄、檢視按鈕與批次回覆目標

## 7. 訊息設計

- 未勾選資料
- 退回缺少意見
- 批次成功 / 失敗清單

## 8. 權限與狀態

- 主管只處理主管階段
- CEO 只處理主管核准後資料
