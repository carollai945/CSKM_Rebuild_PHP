# A03 請假申請 頁面設計

## 1. 基本資料

- 頁面：`A03.jsp`
- 關聯明細頁：`A030.jsp`
- 對應 BDD / SDD / TDD：`a03-leave-request`
- 使用者：申請人

## 2. 頁面目的

- 建立、查詢、修改、送審與取消自己的請假單

## 3. 版面區塊

- 查詢條件區：起迄日期、區域 / 部門、表單狀態、查詢按鈕
- 最近編修提示區：顯示最後一次編修或送審資訊
- 歷史清單區：請假單號、日期、狀態、檢視 / 修改入口
- 建立入口區：新增請假單
- 關聯明細頁區：假別、日期時段、事由、附件與送審操作

## 4. 欄位與顯示規則

- 清單顯示單號、申請日期、假別、狀態與進入明細的按鈕
- 已送審資料在入口頁只保留檢視，不提供草稿編輯
- 草稿或可取消狀態才保留取消 / 刪除入口

## 5. 按鈕與動作

- `建立`：進入新增明細頁
- `儲存`：保存草稿
- `送審`：送入主管審核
- `取消 / 刪除`：僅限可取消狀態

## 6. 互動規則

- 入口頁預設載入最近單據
- 搜尋後以 Ajax 重建結果列表
- 明細頁送出前需檢查日期、假別與事由

## 7. 訊息設計

- 必填欄位未完成
- 日期區間不合法
- 送審成功或刪除成功

## 8. 權限與狀態

- 只可操作自己的單據
- Draft 可編輯，Submitted 後鎖定
